# Basic Usage

## Importing the Image Component

```tsx
import { Image } from '@rixl/videosdk-react';
```

## Essential Props

Every Image component needs at minimum:

```tsx
<Image
  id="your-image-id" // Required: Your image ID from dash.rixl.com
/>
```

## Simple Image Display Example

Here's a basic implementation:

```tsx
import { Image } from '@rixl/videosdk-react';

function BasicImage() {
  return (
    <div className="max-w-2xl mx-auto">
      <Image
        id="your-image-id"
        className="object-contain w-full h-full rounded-lg"
      />
    </div>
  );
}
```

### Image Gallery

```tsx
const galleryImages = [
  { id: 'image-1', alt: 'Gallery image 1' },
  { id: 'image-2', alt: 'Gallery image 2' },
  { id: 'image-3', alt: 'Gallery image 3' },
  { id: 'image-4', alt: 'Gallery image 4' }
];

function ImageGallery() {
  return (
    <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6">
      {galleryImages.map((image) => (
        <div key={image.id} className="aspect-square rounded-lg overflow-hidden">
          <Image
            id={image.id}
            className="w-full h-full object-cover hover:scale-105 transition-transform duration-300"
          />
        </div>
      ))}
    </div>
  );
}
```

### Product Images

```tsx
import { useState } from 'react';

const productImages = [
  { id: 'product-main', alt: 'Main product view' },
  { id: 'product-side', alt: 'Side view' },
  { id: 'product-detail', alt: 'Detail view' },
  { id: 'product-back', alt: 'Back view' }
];

function ProductImages() {
  const [selectedImage, setSelectedImage] = useState(productImages[0]);

  return (
    <div className="grid grid-cols-1 lg:grid-cols-5 gap-4">
      {/* Main Image */}
      <div className="lg:col-span-4 aspect-square">
        <Image
          key={selectedImage.id} // Force re-render on selection change
          id={selectedImage.id}
          className="w-full h-full object-contain rounded-lg"
        />
      </div>

      {/* Thumbnails */}
      <div className="lg:col-span-1 flex lg:flex-col gap-2">
        {productImages.map((image) => (
          <button
            key={image.id}
            onClick={() => setSelectedImage(image)}
            className={`aspect-square rounded-md overflow-hidden border-2 ${
              selectedImage.id === image.id
                ? 'border-blue-500'
                : 'border-transparent hover:border-gray-300'
            }`}
          >
            <Image
              id={image.id}
              className="w-full h-full object-cover"
            />
          </button>
        ))}
      </div>
    </div>
  );
}
```

```tsx

  return (

```

## Best Practices


<Callout type="default">
    **Aspect Ratios**: Use CSS aspect ratio classes to prevent layout shifts during image loading.
</Callout>

<Callout type="warning">
    **Object Fit**: Use `object-contain` to preserve aspect ratio or `object-cover` to fill the container while cropping if necessary.
</Callout>

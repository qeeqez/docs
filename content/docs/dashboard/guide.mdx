This guide will walk you through the essential steps to get started with the RIXL dashboard and begin managing your projects effectively.

## Your First Steps

<Steps>
  <Step>
    **Create a Project**
    
    Go to Settings → Projects → Create a New Project to set up your workspace for media management.
  </Step>
  <Step>
    **Upload Content**
    
    Use the Videos or Images sections to add media content to your projects.
  </Step>
  <Step>
    **Invite Team Members**
    
    Follow the guide in [Organizations → Team Management](/docs/account/organizations/team-management) to add collaborators.
  </Step>
  <Step>
    **Set Up API Access**
    
    Generate authentication keys using [API Access documentation](/docs/account/api-access/create-key) for integrations.
  </Step>
</Steps>

<Callout type="info">
  You need to create a project before accessing media features. Without any projects, you'll be redirected to the projects page.
</Callout>

## Quick Start Workflows

### Upload Your First Video

<Steps>
  <Step>
    **Upload Video**
    
    Click [Upload Video](https://api.rixl.com/videos) in the Videos section and select your files.
  </Step>
  <Step>
    **Configure Settings**
    
    Set basic processing options and quality preferences.
  </Step>
  <Step>
    **Monitor Processing**
    
    Track status: Uploading → Processing → Ready for delivery.
  </Step>
  <Step>
    **Access Content**
    
    Get share URLs, embed codes, and view analytics.
  </Step>
</Steps>

### Upload Your First Image

<Steps>
  <Step>
    **Upload Images**
    
    Click [Upload Image](https://api.rixl.com/images) in the Images section and select your files.
  </Step>
  <Step>
    **Configure Settings**
    
    Set upload preferences and transformation options.
  </Step>
  <Step>
    **Immediate Availability**
    
    Images are instantly available with real-time transformation capabilities.
  </Step>
  <Step>
    **Optimize Delivery**
    
    Use URL-based transformations for automatic optimization and CDN delivery.
  </Step>
</Steps>

## Batch Operations

### Mass Content Management

<Steps>
<Step>
**Select Content**

Use checkboxes or select-all options to choose multiple items
</Step>
<Step>
**Choose Operation**

Select from available batch operations: move, tag, delete, or export
</Step>
<Step>
**Configure Settings**

Set parameters for the operation (folder destination, tags to apply, etc.)
</Step>
<Step>
**Execute and Monitor**

Run the operation with progress tracking and completion notifications
</Step>
</Steps>

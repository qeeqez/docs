## Quality Tiers

Choose the right processing tier for your content during upload:

- **Basic**: Standard web content (720p, efficient processing)
- **Shorts**: Optimized for mobile (1080p, 60fps support)
- **Pro**: High-quality production (4K support, premium features)

<Callout type="info">
  See full details in [Video Processing Quality Tiers](/platform/uploads/videos/quality-tiers).
</Callout>

## Media Library Features

Uploaded videos appear in the **Media Library** with:

- **Smart Categorization** - Automatic sorting by project, date, and content type
- **Thumbnail Management** - Custom thumbnails or auto-generated options
- **Batch Operations** - Bulk editing, moving, and tagging
- **Search & Filter** - Find videos quickly using metadata and processing status

**Dashboard Views:**
- **Processing Queue** - Monitor transcoding progress across all uploads
- **Video Gallery** - Playback previews with actions
- **Analytics View** - Processing time and file size analytics

## Behind the Scenes

All videos are transcoded automatically into multiple qualities with:

- **Tier-based encoding settings**
- **Adaptive streaming support**
- **Optimized compression** for performance
- **Error handling** with retry and failure states

## Dashboard Upload Workflow

### Quick Upload Process

<Steps>
<Step>
**Access Video Upload**

Navigate to your project dashboard → Media → Upload Videos
</Step>
<Step>
**Select Quality Tier**

Choose processing level based on your content needs: Basic, Shorts, or Pro
</Step>
<Step>
**Configure Video Settings**

Set title, description, and project-specific metadata
</Step>
<Step>
**Monitor Processing**

Track transcoding progress with real-time status updates
</Step>
</Steps>

## Quality Tier Selection

Choose the right processing tier for your content during upload:

- **Basic**: Standard web content (720p, efficient processing)
- **Shorts**: Mobile-optimized content (1080p, 60fps support)  
- **Pro**: Professional quality (4K support, premium features)

<Callout type="info">
    For detailed specifications, features, and technical comparisons, see [Video Processing Quality Tiers](/docs/platform/uploads/videos/quality-tiers).
</Callout>

### Media Library Integration

**Organization Features:**
- **Smart Categorization** - Automatic sorting by project, date, and content type
- **Thumbnail Management** - Custom thumbnails or auto-generated options
- **Batch Operations** - Bulk editing, moving, and tagging capabilities
- **Search & Filter** - Find videos quickly using metadata and processing status

**Dashboard Views:**
- **Processing Queue** - Monitor transcoding progress across all uploads
- **Video Gallery** - Visual browsing with playback previews
- **Analytics View** - Processing time and file size analytics

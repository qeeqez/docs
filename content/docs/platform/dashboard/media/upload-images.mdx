  For technical details about formats, optimization, and processing specs, see the [full image upload documentation](/platform/uploads/images).
</Callout>

## Upload Workflows


<Steps>
<Step>
**Access Media Section**

Go to **Media → Images** in your project dashboard.
</Step>
<Step>
**Choose Upload Method**

Choose **Single Upload**, **Batch Upload**, or **Drag & Drop**.
</Step>
<Step>
**Configure Project Settings**

Add metadata (title, tags, folders, project assignment).
</Step>
<Step>
**Monitor Progress**

Track progress with real-time status updates.
</Step>
</Steps>

## Upload Methods

<Tabs items={['Single Upload', 'Batch Upload', 'Drag & Drop']}>
  <Tab value="Single Upload">
    Focused workflow for high-value assets:  
    - Add unique metadata and tags  
    - Assign to a specific project  
    - Adjust individual processing settings  
    </Tab>
    <Tab value="Batch Upload">
    Efficient upload for many files:  
    - Upload multiple images at once  
    - Apply shared settings or metadata  
    - Track overall batch progress  
    </Tab>
    <Tab value="Drag & Drop">
    Fastest way to add files:  
    - Drop images directly from your desktop  
    - Automatic format detection  
    - Upload begins instantly  
  </Tab>
</Tabs>

### Library Features

Once uploaded, images are available in your **Media Library** with the following tools:

- **Smart folders** – Auto-organized by project and date  
- **Tags** – Add custom tags for fast search and filtering  
- **Bulk operations** – Move, update, or delete multiple items  
- **Search & filters** – Find assets quickly by metadata, tags, or dates  

**Dashboard Views:**
- **Grid view** – Thumbnail browsing with quick actions  
- **List view** – Detailed metadata with sorting  
- **Timeline view** – Chronological upload organization

## Behind the Scenes

All uploads are automatically optimized:

- **Format conversion** – WebP/AVIF with fallbacks  
- **Thumbhash** – Lightweight previews for fast UI rendering  
- **Compression & resizing** – Balanced quality and performance  
- **Metadata handling** – Retains essential details for search and SEO  

For deep technical specifications, see the [image upload reference](/platform/uploads/images).

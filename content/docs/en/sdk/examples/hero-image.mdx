## Basic Implementation

```tsx
import { Image } from '@rixl/videosdk-react';

export function HeroSection() {
  return (
    <div className="relative w-full h-[500px]">
      <Image
        id="your-hero-image-id"
        alt="Hero background"
        className="w-full h-full object-cover"
      />
      <div className="absolute inset-0 flex items-center justify-center">
        <h1 className="text-white text-5xl font-bold">Welcome</h1>
      </div>
    </div>
  );
}
```

## Use Cases

### Landing Page Hero

```tsx
<div className="relative w-full h-screen">
  <Image
    id="landing-hero"
    alt="Landing page hero"
    className="w-full h-full object-cover"
  />
  <div className="absolute inset-0 bg-gradient-to-b from-black/60 to-black/20">
    <div className="container mx-auto h-full flex flex-col justify-center px-6">
      <h1 className="text-white text-6xl font-bold mb-4">Your Title Here</h1>
      <p className="text-white/90 text-xl mb-8">Your subtitle here</p>
      <button className="bg-white text-black px-8 py-3 rounded-lg w-fit">
        Get Started
      </button>
    </div>
  </div>
</div>
```

```tsx

```

## Common Props

* `id` (required) - Image ID from dash.rixl.com
* `className` - Tailwind CSS classes for styling


<Callout type="warning">
    **Upload Required**: Upload images to [dash.rixl.com](https://dash.rixl.com) to obtain the required image ID.
</Callout>

## Base URL

Use this base URL for all your API requests:

```
https://api.rixl.com/v1
```

## Quick Start Example

Here are some examples of how to use the RIXL API with the right endpoints:

<Tabs items={['cURL', 'JavaScript', 'Python']}>
<Tab value="cURL">
```bash
# Initialize a video upload
curl -X POST https://api.rixl.com/v1/videos/upload/init \
  -H "X-API-Key: YOUR_PROJECT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "file_name": "my-video.mp4",
    "video_quality": "shorts",
    "image_format": "jpg"
  }'

# Get all videos in your project
curl -X GET https://api.rixl.com/v1/videos \
  -H "X-API-Key: YOUR_PROJECT_API_KEY" \
  -G \
  -d "limit=25" \
  -d "offset=0"

# Get a specific video
curl -X GET https://api.rixl.com/v1/videos/VI9VXQxWXQ \
  -H "X-API-Key: YOUR_PROJECT_API_KEY"
```
</Tab>

<Tab value="JavaScript">
```javascript
// Initialize a video upload
const initResponse = await fetch('https://api.rixl.com/v1/videos/upload/init', {
  method: 'POST',
  headers: {
    'X-API-Key': 'YOUR_PROJECT_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    file_name: 'my-video.mp4',
    video_quality: 'shorts',
    image_format: 'jpg'
  })
});

const uploadData = await initResponse.json();
console.log(uploadData);

// Get all videos in your project
const videosResponse = await fetch('https://api.rixl.com/v1/videos?limit=25&offset=0', {
  headers: {
    'X-API-Key': 'YOUR_PROJECT_API_KEY'
  }
});

const videos = await videosResponse.json();
console.log(videos);
```
</Tab>

<Tab value="Python">
```python
import requests

# Initialize a video upload
response = requests.post(
    'https://api.rixl.com/v1/videos/upload/init',
    headers={
        'X-API-Key': 'YOUR_PROJECT_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'file_name': 'my-video.mp4',
        'video_quality': 'shorts',
        'image_format': 'jpg'
    }
)

upload_data = response.json()
print(upload_data)

# Get all videos in your project
videos_response = requests.get(
    'https://api.rixl.com/v1/videos',
    headers={'X-API-Key': 'YOUR_PROJECT_API_KEY'},
    params={'limit': 25, 'offset': 0}
)

videos = videos_response.json()
print(videos)
```
</Tab>
</Tabs>

## Response Format

API responses follow standard REST patterns:

### Video Upload Init Response

<TypeTable
  type={{
    video_id: {
      description: 'Unique identifier for the uploaded video',
      type: 'string',
      default: 'VI9VXQxWXQ'
    },
    video_presigned_url: {
      description: 'Pre-signed URL for uploading the video file',
      type: 'string'
    },
    poster_id: {
      description: 'Unique identifier for the video poster/thumbnail',
      type: 'string',
      default: 'PS5IMKoFLm'
    },
    poster_presigned_url: {
      description: 'Pre-signed URL for uploading the poster image',
      type: 'string'
    },
    upload_expires: {
      description: 'Unix timestamp when the upload URLs expire',
      type: 'number',
      default: 1640995200
    }
  }}
/>

**Example Response:**
```json
{
  "video_id": "VI9VXQxWXQ",
  "video_presigned_url": "https://videos.rixl.com/Bq4y3QB38S/VI9VXQxWXQ?X-Amz-Algorithm=...",
  "poster_id": "PS5IMKoFLm", 
  "poster_presigned_url": "https://images.rixl.com/Bq4y3QB38S/PS5IMKoFLm?X-Amz-Algorithm=...",
  "upload_expires": 1640995200
}
```

**Paginated Response (List Videos):**
```json
{
  "data": [
    {
      "id": "VI9VXQxWXQ",
      "duration": 30.5,
      "width": 1920,
      "height": 1080,
      "file": {
        "id": "FpvLwyDMqu",
        "name": "original.mp4",
        "size": 1048576,
        "url": "https://videos.rixl.com/Bq4y3QB38S/FpvLwyDMqu",
        "status": "ready"
      }
    }
  ],
  "pagination": {
    "limit": 25,
    "offset": 0,
    "total": 100
  }
}
```

## Error Handling

When an error occurs, the API returns a structured error response:

<TypeTable
  type={{
    code: {
      description: 'HTTP status code indicating the type of error',
      type: 'number',
      default: 400
    },
    error: {
      description: 'Brief description of the error',
      type: 'string',
      default: 'Invalid request body'
    },
    details: {
      description: 'Additional context about the error',
      type: 'string',
      default: 'The provided file format is not supported'
    }
  }}
/>

**Example Error Response:**

```json
{
  "code": 400,
  "error": "Invalid request body",
  "details": "The provided file format is not supported"
}
```

## Rate Limits

<Callout type="warning">
Each subscription plan has its own API call limits. If you reach your limit, new requests will be denied until your usage resets. You can check your usage in your dashboard to avoid any interruptions.
</Callout>

| Plan | Requests per Hour | Burst Limit |
|------|-------------------|-------------|
| Free | 100 | 10 per minute |
| Pro | 1,000 | 50 per minute |
| Enterprise | Custom | Custom |

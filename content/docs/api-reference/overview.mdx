## Base URL

All API requests should be made to:

```
https://api.rixl.com
```

## Authentication

To access the RIXL API, add your API key to the Authorization header:

```bash
Authorization: YOUR_PROJECT_API_KEY
```

## Quick Start Example

Here's how you can work with RIXL's API using the correct endpoints:

<Tabs items={['cURL', 'JavaScript', 'Python']}>
<Tab value="cURL">
```bash
# Initialize a video upload
curl -X POST https://api.rixl.com/videos/upload/init \
  -H "X-API-KEY: YOUR_PROJECT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "file_name": "my-video.mp4",
    "video_quality": "shorts",
    "image_format": "jpg"
  }'

# Get all videos in your project
curl -X GET https://api.rixl.com/videos \
  -H "X-API-KEY: YOUR_PROJECT_API_KEY" \
  -G \
  -d "limit=25" \
  -d "offset=0"

# Get a specific video
curl -X GET https://api.rixl.com/videos/VI9VXQxWXQ \
  -H "X-API-KEY: YOUR_PROJECT_API_KEY"
```
</Tab>

<Tab value="JavaScript">
```javascript
// Initialize a video upload
const initResponse = await fetch('https://api.rixl.com/videos/upload/init', {
  method: 'POST',
  headers: {
    'X-API-KEY': 'YOUR_PROJECT_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    file_name: 'my-video.mp4',
    video_quality: 'shorts',
    image_format: 'jpg'
  })
});

const uploadData = await initResponse.json();
console.log(uploadData);

// Get all videos in your project
const videosResponse = await fetch('https://api.rixl.com/videos?limit=25&offset=0', {
  headers: {
    'X-API-KEY': 'YOUR_PROJECT_API_KEY'
  }
});

const videos = await videosResponse.json();
console.log(videos);
```
</Tab>

<Tab value="Python">
```python
import requests

# Initialize a video upload
response = requests.post(
    'https://api.rixl.com/videos/upload/init',
    headers={
        'X-API-KEY': 'YOUR_PROJECT_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'file_name': 'my-video.mp4',
        'video_quality': 'shorts',
        'image_format': 'jpg'
    }
)

upload_data = response.json()
print(upload_data)

# Get all videos in your project
videos_response = requests.get(
    'https://api.rixl.com/videos',
    headers={'X-API-KEY': 'YOUR_PROJECT_API_KEY'},
    params={'limit': 25, 'offset': 0}
)

videos = videos_response.json()
print(videos)
```
</Tab>
</Tabs>

## Response Format

Based on your API specification, responses follow standard REST patterns:

**Successful Response (Video Upload Init):**
```json
{
  "video_id": "VI9VXQxWXQ",
  "video_presigned_url": "https://videos.rixl.com/Bq4y3QB38S/VI9VXQxWXQ?X-Amz-Algorithm=...",
  "poster_id": "PS5IMKoFLm", 
  "poster_presigned_url": "https://images.rixl.com/Bq4y3QB38S/PS5IMKoFLm?X-Amz-Algorithm=...",
  "upload_expires": 1640995200
}
```

**Paginated Response (List Videos):**
```json
{
  "data": [
    {
      "id": "VI9VXQxWXQ",
      "duration": 30.5,
      "width": 1920,
      "height": 1080,
      "file": {
        "id": "FpvLwyDMqu",
        "name": "original.mp4",
        "size": 1048576,
        "url": "https://videos.rixl.com/Bq4y3QB38S/FpvLwyDMqu",
        "status": "ready"
      }
    }
  ],
  "pagination": {
    "limit": 25,
    "offset": 0,
    "total": 100
  }
}
```

## Error Handling

Error responses follow the ResponseError schema:

```json
{
  "code": 400,
  "error": "Invalid request body",
  "details": "The provided file format is not supported"
}
```

## Rate Limits

<Callout type="warning">
Your subscription plan sets specific limits on the number of API calls you can make. If you reach your rate limit, additional requests may be denied until your usage resets. Monitor your API usage in your dashboard to avoid interruptions.
</Callout>

| Plan | Requests per Hour | Burst Limit |
|------|-------------------|-------------|
| Free | 100 | 10 per minute |
| Pro | 1,000 | 50 per minute |
| Enterprise | Custom | Custom |

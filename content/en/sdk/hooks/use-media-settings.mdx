---
title: "useMediaSettings"
description: "Read and update player-scoped mute and volume state for a specific Video instance."
---

`useMediaSettings(playerId)` exposes player-scoped audio state and actions.

## Signature

```tsx
const media = useMediaSettings(playerId);
```

It returns:

- `muted`
- `volume`
- `previousVolume`
- `mute()`
- `unmute()`
- `toggleMute()`
- `setVolume(volume)`

## Example

```tsx
import {Video, useMediaSettings} from "@rixl/videosdk-react";

const playerId = "00000000-0000-0000-0000-0000000000c3";

function CustomVolumeControls() {
  const {muted, volume, toggleMute, setVolume} = useMediaSettings(playerId);

  return (
    <div className="space-y-3">
      <Video
        id="your-video-id"
        playerId={playerId}
        theme="hideUI"
        analytics={false}
      />

      <div className="flex items-center gap-3">
        <button type="button" onClick={() => toggleMute()}>
          {muted ? "Unmute" : "Mute"}
        </button>

        <input
          type="range"
          min={0}
          max={1}
          step={0.05}
          value={volume}
          onChange={(event) => setVolume(Number(event.target.value))}
        />
      </div>
    </div>
  );
}
```

## Notes

- The hook only affects the player identified by `playerId`.
- `muted` reflects the effective audible state, including volume `0`.
- Use [`useGlobalPlayerSettings()`](/en/sdk/hooks/use-global-player-settings) when one control should fan out to every mounted player on the page.

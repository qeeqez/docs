# Basic Usage

## Importing the Image Component

```tsx
import { Image } from '@rixl/videosdk-react';
```

## Two Ways to Use Image Component

The Image component supports two usage patterns:

### 1. **Fetch by ID** (Recommended for standalone images)

```tsx
<Image
  id="your-image-id" // Fetches automatically
  className="w-full h-full object-cover"
/>
```

### 2. **Pre-fetched Data** (Recommended for feeds/galleries)

```tsx
<Image
  imageData={imageData} // Already fetched
  className="w-full h-full object-cover"
/>
```

<Callout type="info">
  Use `id` when displaying a single image. Use `imageData` when rendering multiple images from a feed or batch API call to avoid redundant fetches.
</Callout>

## Simple Image Display Example

Here's a basic implementation:

```tsx
import { Image } from '@rixl/videosdk-react';

function BasicImage() {
  return (
    <div className="max-w-2xl mx-auto">
      <Image
        id="your-image-id"
        alt="Product showcase"
        className="object-contain w-full h-full rounded-lg"
      />
    </div>
  );
}
```

### Image Gallery (with ID)

```tsx
const galleryImages = [
  { id: 'image-1', alt: 'Gallery image 1' },
  { id: 'image-2', alt: 'Gallery image 2' },
  { id: 'image-3', alt: 'Gallery image 3' },
  { id: 'image-4', alt: 'Gallery image 4' }
];

function ImageGallery() {
  return (
    <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6">
      {galleryImages.map((image) => (
        <div key={image.id} className="aspect-square rounded-lg overflow-hidden">
          <Image
            id={image.id}
            alt={image.alt}
            className="w-full h-full object-cover hover:scale-105 transition-transform duration-300"
          />
        </div>
      ))}
    </div>
  );
}
```

### Feed with Pre-fetched Data

```tsx
function FeedImages({ posts }) {
  const [currentIndex, setCurrentIndex] = useState(0);
  
  return (
    <div className="space-y-4">
      {posts.map((post, index) => (
        <Image
          key={post.id}
          imageData={post.image}
          isCurrent={index === currentIndex}
          feedId="homepage-feed"
          postId={post.id}
          alt={post.title}
          className="w-full aspect-video object-cover"
        />
      ))}
    </div>
  );
}
```

### Loading States with Callback

```tsx
function ImageWithLoader() {
  const [isLoading, setIsLoading] = useState(true);
  
  return (
    <div className="relative">
      {isLoading && (
        <div className="absolute inset-0 bg-gray-200 animate-pulse rounded-lg" />
      )}
      <Image
        id="your-image-id"
        onLoadingChange={setIsLoading}
        className="w-full h-full object-cover rounded-lg"
      />
    </div>
  );
}
```

## Best Practices

<Callout type="default">
  **Aspect Ratios**: Use CSS aspect ratio classes to prevent layout shifts during image loading.
</Callout>

<Callout type="warning">
  **Object Fit**: Use `object-contain` to preserve aspect ratio or `object-cover` to fill the container while cropping
  if necessary.
</Callout>

<Callout type="info">
  **Pre-fetched Data**: When rendering multiple images (galleries, feeds), fetch all data in one API call and pass `imageData` to avoid N+1 query problems.
</Callout>

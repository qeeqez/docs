The Feed component provides a professional-grade, infinite-scrolling feed experience optimized for social media platforms. It delivers smooth vertical scrolling with automatic video playback, intelligent content loading, and touch gesture support that works seamlessly across all modern browsers and devices.

### Key Capabilities

**Scrolling & Content Loading**
- Infinite vertical scrolling with CSS snap scrolling for TikTok-style navigation
- 5-component carousel recycling for memory-efficient rendering
- Next video preloaded automatically, distant videos (5+ positions away) unloaded
- New content fetched automatically when within 3 posts of the end
- Posts cached locally in localStorage (up to 50 posts)

**Media Playback**
- Videos **loop by default** and start **muted** at **0.5 volume**
- Auto-unmute on first user interaction
- Automatic play/pause based on scroll position (current post plays, others pause)
- Fullscreen is disabled in feed
- Support for both video and image posts

**Post Descriptions**
- 2-line truncation by default with "More/Less" toggle
- Expands up to 70% viewport height when toggled
- Automatically collapses when scrolling to the next post

**Image Posts**
- Thumbhash placeholder shown while loading
- Smooth fade-in transition on load
- Aspect ratio preserved (contain for wide images, cover for tall)

**User Interaction**
- Touch gesture support via native scroll snapping
- Keyboard shortcuts (Space to play/pause, M to mute, arrows for volume)
- Responsive design with mobile-first approach

**Performance**
- 5-component carousel recycles DOM elements for memory efficiency
- Preload states: current video buffers fully, next 1 preloads, rest load metadata only
- Passive scroll listeners for smooth performance
- Async image decoding

### When to Use

The Feed component is perfect for:
- **Social media platforms** requiring vertical scrolling feeds
- **Content discovery applications** with video and image posts
- **Creator platforms** with user-generated content
- **Mobile-first apps** requiring TikTok-style experiences
- **Community platforms** with feed-based content distribution

### Browser Support

Works seamlessly across all modern browsers with automatic fallbacks. Optimized for both mobile and desktop experiences while maintaining core functionality across all devices.
